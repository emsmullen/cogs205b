# cogs 205b hw 4: test-driven development
Instrctions for using bayes factor test suite

## directions

This repo is setup to use the Bayes Factor test suite. 

1. Build a container using the Dockerfile and attach your VS Code session to the running container
2. Create a virtual environment using 'python3 -m venv ./path-to-new-venv' then use pip to install the required packages listed in requirements.txt
4. Make sure you are in the directory named bayes_factor
5. To run the tests, use the command 'python -m unittest tests/test_bayes_factor.py'
